# Query Honcho Memory

Ask Honcho what it knows about you from previous conversations.

## Usage

```
/honcho-query "What are my coding preferences?"
```

## Examples

```
/honcho-query "What editor do I prefer?"
/honcho-query "What database decisions did we make?"
/honcho-query "Tell me about my work background"
```

## What It Returns

A synthesized answer based on observations extracted from your stored messages.

**Note:** Works best after ~1 minute of storing messages (deriver processing time).
